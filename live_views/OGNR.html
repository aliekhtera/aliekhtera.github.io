Autonomous Object-Goal Navigation Robot

This project was conducted at the Software-Defined Communication Networks Lab, Sharif University of Technology, under the supervision of Prof. Babak Khalaj. The following report provides a detailed description of my Bachelor’s Thesis and its technical contributions.

My Bachelor's thesis focused on developing and implementing a comprehensive framework for an Autonomous Robotic Agent, specifically designed for efficient navigation and language-grounded object retrieval within unknown indoor environments. To ensure cost-effectiveness, We engineered a novel 3D mapping subsystem based on Stereo Vision and advanced feature matching (e.g., Roma), thereby minimizing the need for expensive LiDAR sensors or high-capacity computing hardware. For robust autonomous movement, the agent employs a frontier-based exploration strategy for target selection, integrated with the RRT* algorithm to guarantee fast and collision-free path planning.

The framework incorporates a hierarchical object retrieval pipeline that leverages state-of-the-art Vision-Language Models (VLMs), including CLIP and Grounding-DINO, within a refined architecture to accurately localize and identify target objects based solely on natural language commands.

As a continuation of this project, I developed improved methods for 3D Semantic Segmentation, as well as for Object Retrieval and Natural Language Command Processing. These advancements have been formalized into a research paper that is currently under review for ICLR.
